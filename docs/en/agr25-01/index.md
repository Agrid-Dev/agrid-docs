# AGR25-01 Fan Coil Thermostat

The AGR25-01 is a modern fan coil thermostat combining intuitive touchscreen control with advanced WiFi connectivity. Designed for compatibility with 2-pipe and 4-pipe HVAC systems, it offers flexible configuration options suitable for residential and commercial installations.

## Key Features

- **4-inch Color Touchscreen** - Clear, responsive interface for temperature and mode control
- **WiFi 2.4GHz Connectivity** - Remote management via AGRID mobile application
- **Flexible Relay Outputs** - 5× SPST-NO relays (5A max) for fan and valve control
- **Analog Control** - 3× DAC 0-10V outputs for proportional control
- **Dual Sensors** - Support for analog (thermistor) and digital (dry contact) sensors
- **Autonomous Operation** - Works independently if WiFi connection is lost
- **Compact Design** - Wall-mounted unit (105×85×30mm) fits standard wall boxes
- **Energy Efficient** - Low power consumption (MAX 2W operating)
- **Safety-Certified** - Class II, IP20 rated, CE and RoHS compliant

## Operating Modes

### Heat Mode
Controls heating via proportional valve control (0-10V) or ON/OFF valve switching with adaptive fan speed. Maintains setpoint temperature during heating periods.

### Cool Mode
Controls cooling via proportional valve control or ON/OFF valve switching with adaptive fan speed. Optimizes comfort during cooling operation.

### Fan Only Mode
Controls fan speed independent of heating or cooling. Useful for air circulation without temperature control.

### Auto Mode
Automatically switches between heating and cooling based on room temperature and setpoint. Intelligent mode selection optimizes comfort and energy efficiency.

## System Configurations

The AGR25-01 supports multiple HVAC configurations:

- **2-Pipe Systems** - Single valve with seasonal switching between heating and cooling
- **4-Pipe Systems** - Separate heating and cooling valves for simultaneous availability
- **Electric Heating** - Integration of electric heater element via external contactor (relay-controlled)
- **Proportional Control** - 0-10V analog output for advanced valve control
- **On/Off Control** - Relay-based simple switching for basic installations

Detailed wiring configurations and examples are provided in the [Wiring Configurations](wiring.md) section.
