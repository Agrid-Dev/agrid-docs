# Usage

## Local Control via Touchscreen

From the touchscreen, the user can:

- **Turn on/off:** Enable or disable the thermostat
- **Adjust setpoint temperature:** Set desired temperature within the configurable range (5°C to 35°C)
- **Select mode:** Heat / Cool / Fan only / Auto
- **Select fan speed:** Low / Medium / High / Auto

## Remote Control via WiFi

The thermostat is also controllable remotely via WiFi 2.4 GHz and AGRID app.

!!! note "Autonomous Operation"
    In case of WiFi loss, thermostat continues in local autonomous mode.

## Security and Data

### WiFi Configuration

WiFi credentials are configured during commissioning via the AGRID Installer app. Each thermostat has unique credentials.

### Firmware Updates

Software updates are performed remotely (OTA) via the AGRID server securely.

### Factory Reset

A complete reset is possible from the settings screen or via the AGRID app. This erases all personal data (WiFi credentials, configuration) and restores factory settings.
