# AGRID Thermostat Documentation

Welcome to the official AGRID thermostat documentation. This site contains comprehensive information about installation, configuration, usage, and technical specifications for AGRID climate control products.

## Product Overview

AGRID thermostats are advanced fan coil controllers designed for modern HVAC systems. With WiFi connectivity, intuitive touchscreen interfaces, and flexible control configurations, AGRID thermostats offer reliable performance for residential and commercial applications.
