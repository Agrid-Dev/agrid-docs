# AGR25-01 - Thermostat pour ventilo-convecteurs

## Présentation du produit

Le thermostat **AGR25-01** est un contrôleur multifonctions conçu pour la régulation précise de ventilo-convecteurs en applications de climatisation et chauffage. Doté d'un écran tactile couleur et d'une connectivité WiFi intégrée, il offre une gestion locale et à distance du confort thermique.

| **Caractéristique** | **Spécification** |
|---|---|
| **Modèle** | AGR25-01 |
| **Fabricant** | Beijing Breeze Technology Co., Ltd. |
| **Importateur** | AGRID SAS (Paris) |
| **Classe de sécurité** | Classe II |
| **Protection** | IP20 |
| **Indice de température** | T50 |
| **Type de coupure** | Type 1.B micro-coupure |

|---|---|
| **Alimentation** | 220-240V~ 50Hz |

## Caractéristiques fonctionnelles

| **Fonction** | **Description** |
|---|---|
| **Connectivité** | WiFi 2.4GHz pour pilotage à distance |
| **Interface** | Écran tactile couleur 4 pouces |
| **Régulation locale** | Contrôle via interface tactile ou entrées capteurs |
| **Régulation à distance** | Application AGRID Installer |
| **Mode autonome** | Fonctionnement indépendant en cas de perte WiFi |
| **Endurance mécanique** | 100 000 cycles automatiques / 6 000 manuels |
