# Utilisation

## Commande locale via l'écran tactile

Depuis l'écran tactile, l'utilisateur peut :

- Allumer / éteindre le thermostat (**ON / OFF**)
- Régler la **consigne de température**
- Sélectionner le mode de fonctionnement
- Sélectionner la vitesse du ventilateur

### Modes de fonctionnement

| Mode | Description |
|---|---|
| **Heat** | Chauffage uniquement — le thermostat commande les sorties de chauffage pour atteindre la consigne |
| **Cool** | Refroidissement uniquement — le thermostat commande les sorties de refroidissement pour atteindre la consigne |
| **Fan** | Ventilation seule — pas de chauffage ni de refroidissement, le ventilateur tourne à la vitesse sélectionnée |
| **Auto** | Commutation automatique entre Heat et Cool en fonction de la température ambiante et de la consigne |

### Vitesse du ventilateur

| Vitesse | Description |
|---|---|
| **Low** | Ventilateur en vitesse basse |
| **Medium** | Ventilateur en vitesse moyenne |
| **High** | Ventilateur en vitesse haute |
| **Auto** | Vitesse ajustée automatiquement par le thermostat en fonction de l'écart entre la température ambiante et la consigne |

## Commande à distance via WiFi

Le thermostat peut également être commandé à distance via WiFi 2,4 GHz et l'application AGRID.

!!! note "Fonctionnement autonome"
    En cas de perte de connexion WiFi, le thermostat continue de fonctionner en mode local autonome.

## Sécurité et données

### Configuration WiFi

Les identifiants WiFi sont configurés lors de la mise en service via l'application AGRID Installer.

### Mises à jour du firmware

Les mises à jour logicielles sont effectuées à distance (OTA) via le serveur AGRID de manière sécurisée.

### Réinitialisation usine

Une réinitialisation complète du thermostat est possible depuis l'écran des paramètres ou via l'application AGRID. Cette opération efface toutes les données personnelles (identifiants WiFi, configuration) et restaure les réglages d'usine.
